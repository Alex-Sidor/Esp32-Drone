# Installation
Download all the libraries below into arduino ide

* Adafruit mpu6050
* ESP32ServoController

my drone is comprised of
* 4x 8520 brushless dc motors
* 4x props (came with dc motors)
* 2x MX1508 DC Motor Driver Module
* 1x Esp32 Devkit V1 (you can use whatever esp you want)
* 1x mpu6050
* 1x 3.7v battery (max 4.2v)
* assorted wire
* assorted capacitors (read on where to put them)
* rubber spacers to remove vibration from the mpu6050 (get better readings)
* long m2 bolts and nuts to secure top part

for the drone frame i used

`https://www.thingiverse.com/thing:1221911`

I wanted a quick print that i knew was going to work so i didn't design my own one
